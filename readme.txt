开发环境：Eclipse+Tomcat7.0+Maven
软件架构：SSM +Redis+Nginx+RabbitMQ+Mysql
项目描述：该项目是一个综合性的电子购物平台，实现了后台系统、前台系统、购物车系统、订单系统、搜索系统、物流系统以及支付系统。顾客可以通过前台系统访问商品，同时利用购物车系统、订单系统、支付系统完成购买商品的流程；可以利用搜索系统搜索商品，利用物流系统查看商品的物流信息等。
技术描述：
1.	利用Maven对项目进行继承、依赖管理，并垂直分布模块
2.	使用EasyUI框架搭建后台管理界面，使用ajax技术访问后台程序
3.	使用Nginx服务器实现动静分离、负载均衡
4.	使用HttpClient技术实现异构系统之间的通信
5.	使用Redis集群，提高查询的效率
6.	使用SSO单点登录
7.	RabbitMQ消息队列减少系统并发压力

个人描述：使用dubbo框架重构订单系统和购物车系统
